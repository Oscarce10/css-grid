Grid CSS Cuarta parte

Si abrimos la consola de developer tools podemos encontrar unos numeros denominados tracks que marcan el comienzo y el fin de cada fila y columna
Adicionalmente, se crean varias lineas unas continuas y otras punteadas
  • La línea cortada vertical la entendemos como un explicit grid que son definidad con las propiedades de grid-template-columns y grid-template-rows
  • Las líneas en diagonal se entienden como el gap
  • La línea punteada horizontal se entiende como inplicit grid ya que no las declaramos propiamente y son generadas por el browser
  • La línea vertical solida se entiende como el comienzo y el final del grid
Podemos definir el tamaño de las columnas o filas que se agregan de manera inplicita por medio de grid-auto-rows ó grid-auto-columns
